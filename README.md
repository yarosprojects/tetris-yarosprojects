# Tetris Game

A classic Tetris game built with **Svelte**, **TypeScript**, and **HTML5 Canvas**. Play Tetris directly in your browser with keyboard controls or touch-friendly mobile buttons.

## Features

- Classic Tetris gameplay
- Keyboard controls (Arrow keys)
- Mobile-friendly on-screen controls
- Score tracking
- Game over detection
- Smooth animations with canvas rendering
- Rotating and moving Tetris pieces
- Fully responsive design

## Technologies Used

- **Svelte** – Frontend framework for building reactive UI
- **TypeScript** – Strongly typed JavaScript
- **HTML5 Canvas** – Rendering the game board
- **JavaScript** – Game logic and interactivity
- **TailwindCSS** – Styling and responsive design

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yarosprojects/tetris.git
cd tetris
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser at http://localhost:5173 (or the port shown in terminal).
